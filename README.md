# How to Run the android_test Project

Follow these steps to set up and run the `android_test` project on your local machine.

## Prerequisites

- **Android Studio**: Ensure you have the latest version installed. You can download it from the [official Android Developers website](https://developer.android.com/studio).

- **Git**: Required for cloning the repository. Download it from the [official Git website](https://git-scm.com/downloads) if you don't have it installed.

## Steps to Run the Project

1. **Clone the Repository**

   Open your terminal or command prompt and execute:

   ```bash
   git clone https://github.com/VHFroesVianna/android_test.git
   cd android_test
# Open the Project in Android Studio

Launch Android Studio.
Click on "Open an existing Android Studio project".
Navigate to the cloned android_test directory and select it.
Sync Project with Gradle Files

Once the project is open:

Click on "File" > "Sync Project with Gradle Files" to ensure all dependencies are downloaded and configured correctly.
Build the Project

Navigate to "Build" > "Make Project" or press Ctrl+F9 (or Cmd+F9 on macOS) to build the project.
Run the Project

Ensure you have an Android device connected or an emulator running.
Click on the "Run" button (green arrow) in the toolbar or press Shift+F10 to start the application.
